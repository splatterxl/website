Telegram

Email

To send me an email, you can write to splatterxl [at] outlook.ie. To send me an encrypted email, use my PGP public key and attach a copy of your own public key so I can respond accordingly.

Many more

I can be found on some other services as Splatterxl. If I don't respond within 3 months, I've probably forgotten about that service. If this happens, simply try another.

Security Vulnerabilities

If you ever find a security vulnerability in any of my projects, DO NOT MAKE AN ISSUE IN THE REPOSITORY! Instead, send me an email with an appropriate subject, eg. Security issue in Lightbulb. Include all necessary details such as reproduction steps and your opinion on its severity. If possible, encrypt your email as well.
